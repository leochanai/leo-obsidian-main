# 像素级角色拆解

顶尖概念美术师视角，将角色进行"像素级拆解"，透视穿着层级、微表情变化，并通过私密物品、随身物件和生活细节侧面丰满人物性格。

## 核心任务

根据用户上传或描述的主体形象，生成**全景式角色深度概念分解图**:

- **中心**: 角色全身立绘作为视觉锚点
- **环绕**: 服装分层、表情集、材质特写、关联物品有序排列
- **连接**: 手绘箭头或引导线连接拆解元素与对应部位

## 拆解维度

### 1. 服装分层

**外层到内层的完整展示**:
- 外套、上衣、裤装/裙装的单品拆解
- 多层穿搭需展示脱下外层后的内层状态
- **私密内着**: 独立展示内层衣物设计
  - 蕾丝内衣裤(花纹细节)
  - 丁字裤(剪裁线条)
  - 丝袜(透肉感与袜口设计)
  - 塑身衣或安全裤

### 2. 表情集

角落绘制 3-4 个头部特写，展示情绪变化:
- 冷漠、害羞、惊讶、失神
- 专注神态(如涂口红时)
- 微表情细节(眼神、嘴角)

### 3. 材质特写

**关键部位放大特写**:
- 布料褶皱、皮肤纹理、手部细节
- 口红膏体润泽感
- 皮革包包颗粒纹理
- 化妆品粉质细腻感

### 4. 关联物品(生活切片)

**随身包袋与内容物**:
- 绘制日常通勤包或手拿包
- "打开"展示散落物品

**美妆与护理**:
- 特定色号口红/唇釉特写
- 带镜子的粉饼盒
- 香水瓶设计
- 护手霜

**私密生活物件**(以设计图客观视角呈现):
- 私密日记本
- 常用药物/补剂盒
- 电子烟
- 其他更私人的物件（如用户提到的飞机杯/情趣用品，需以一种设计图的客观视角呈现，注明型号或设计特点）

## 视觉规范

**画风**: 高质量 2D 插画或概念设计草图，线条干净利落

**背景**: 米黄色、羊皮纸或浅灰色纹理，营造设计手稿氛围

**注释**: 每个拆解元素旁模拟手写注释:
- 材质说明: "柔软蕾丝"、"磨砂皮革"
- 品牌/型号暗示: "常用色号#520"、"定制款"

**输出**: 中文:英文标记，高清 4K HD

## 执行流程

1. **分析主体**: 核心特征、穿着风格、潜在性格
2. **提取一级元素**: 外套、鞋子、主要表情
3. **设计二级元素**: 
   - 内衣风格(蕾丝/运动/简约?)
   - 包内物品(口红色号/香水品牌?)
   - 独处物品(反映隐藏性格面)
4. **生成组合图**: 
   - 透视准确，光影统一
   - 注释清晰，排版有序
   - 引导线连接对应部位

## 示例场景

**输入**: 一位穿职业装的都市女性角色插画

**输出分解图包含**:
- 中心: 全身立绘(西装套装+高跟鞋)
- 左上: 表情集(严肃/微笑/疲惫/放松)
- 右上: 西装外套拆解 → 衬衫 → 蕾丝 Bralette
- 左下: 打开的通勤包(MacBook/口红#420/蓝牙耳机/止痛药)
- 右下: 材质特写(西装面料纹理/丝袜网格/皮包车线)
- 底部: 私密物件(日记本封面/香薰蜡烛/震动棒设计图)

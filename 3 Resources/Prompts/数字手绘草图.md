# 数字手绘草图

![[_Attachment/03d4cc0066ab8307199381a60ad40d07_MD5.jpg]]

```md
主题：电影《鬼妈妈》

一张[主题]的数字手绘草图。风格松散、随意，具有一种“画工拙劣”的涂鸦美学，黑色轮廓线粗糙、颤抖且不均匀。关键在于，要在人物和物体内部加入大量潦草、扭曲的黑色线条，以表现衣物褶皱、弄皱的布料、头发纹理和建筑细节，尽管缺乏写实感，却能营造出一种繁杂且独特的线条纹理。 采用实色色块的平涂上色，赛璐珞风格阴影（Cel-shaded），无渐变，无写实光影。极简的面部特征（眼睛只是简单的圆点，或者是无五官的面孔）。背景为平涂的米色或暖米白色。一种幽默、低保真、稚拙的数字插画风格。 3:2
```

# A 股每日复盘

从东方财富、同花顺等财经网站抓取当日市场数据和新闻，生成结构化的 Markdown 复盘报告。

## 工作流

复制此清单并跟踪进度：

```
任务进度：
- [ ] 步骤 1：获取市场数据
- [ ] 步骤 2：分析市场行情
- [ ] 步骤 3：生成 Markdown 报告
```

---

## 步骤 1：获取市场数据

### 数据来源

按优先级从以下网站抓取信息：

| 来源 | URL | 主要内容 |
|------|-----|----------|
| **东方财富** | https://www.eastmoney.com/ | 大盘指数、热点板块、资金流向 |
| **东方财富股吧** | https://guba.eastmoney.com/ | 市场情绪、热门讨论 |
| **同花顺财经** | https://www.10jqka.com.cn/ | 行情综述、龙头股分析 |
| **新浪财经** | https://finance.sina.com.cn/stock/ | 财经新闻、政策解读 |
| **雪球** | https://xueqiu.com/ | 专业投资者观点 |

### 抓取要点

使用 `read_url_content` 工具依次抓取：

1. **大盘指数**：上证指数、深证成指、创业板指、科创50
2. **涨跌统计**：涨停/跌停数量、上涨/下跌家数
3. **成交额**：两市总成交额、北向资金净流入
4. **热点板块**：涨幅前5板块、跌幅前5板块
5. **龙头个股**：连板股、人气股、妖股
6. **重要新闻**：政策面、资金面、消息面

---

## 步骤 2：分析市场行情

基于抓取的数据，进行专业分析：

### 2.1 大盘分析

- 三大指数表现（涨跌幅、成交量变化）
- 技术面分析（支撑/压力位、均线趋势）
- 量能分析（缩量/放量、量价配合）

### 2.2 板块分析

- 主线逻辑：当日领涨板块的驱动因素
- 资金偏好：主力资金流入板块
- 板块轮动：与前一日对比的变化

### 2.3 市场情绪

- 赚钱效应：涨停家数、连板高度
- 亏钱效应：跌停家数、炸板率
- 市场人气：成交额、换手率

### 2.4 后市展望

- 短期关注点
- 潜在风险
- 操作建议（仅供参考，不构成投资建议）

## 常见问题

### Q: 网站无法访问或数据不完整？

优先使用可访问的数据源，报告中标注数据缺失情况。

### Q: 如何获取实时数据？

本 Skill 基于网页抓取，数据存在延迟。如需实时数据，建议在交易日收盘后（15:30 之后）执行。

### Q: 报告日期如何确定？

默认使用当前日期。如需复盘历史日期，用户需明确指定日期。

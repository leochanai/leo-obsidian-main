## 填写指南

| 字段 | 数据来源 | 备注 |
|------|----------|------|
| 大盘指数 | 搜索结果 | 收盘价、涨跌幅、成交额 |
| 涨跌统计 | 搜索结果 | 涨停/跌停家数 |
| 热点板块 | 搜索结果 | 领涨/领跌前3 |
| 连板龙头 | 搜索结果 | 最高连板股 |
| 资金流向 | 搜索结果 | 北向资金、主力流向 |
| 重要消息 | 搜索结果 | 当日重大新闻 |
| 后市展望 | AI 分析 | 基于数据的研判 |

如某项数据无法获取，在报告中标注"数据暂缺"。

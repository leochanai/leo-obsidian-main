# 模板占位符速查表

## 📋 基础信息

| 占位符 | 说明 | 示例 |
|--------|------|------|
| `{{DATE}}` | 报告日期 | `2026-01-12` |
| `{{MARKET_SUMMARY}}` | 市场概述 | `沪指17连阳创历史 成交额3.6万亿创新高` |
| `{{RISK_BANNER}}` | 风险横幅（可选） | 见模板 |

## 📊 内容区占位符

| 占位符 | 对应章节 | HTML 组件 |
|--------|---------|-----------|
| `{{INDEX_CARDS}}` | 大盘概览 | 指数卡片（上证/深证/创业板） |
| `{{VOLUME_STATS}}` | 大盘概览 | 成交量/北向资金统计 |
| `{{STAT_CARDS}}` | 涨跌统计 | 上涨/下跌/涨停/封板率卡片 |
| `{{HOT_SECTORS_TABLE}}` | 热点板块 | 领涨板块表格 |
| `{{COLD_SECTORS_TABLE}}` | 热点板块 | 领跌板块表格 |
| `{{LEADER_CARDS}}` | 连板龙头 | 龙头股卡片 |
| `{{CAPITAL_FLOW_CARDS}}` | 资金流向 | 资金流向分类卡片 |
| `{{NEWS_CARDS}}` | 重要消息 | 新闻时间线卡片 |
| `{{OUTLOOK_PANELS}}` | 后市展望 | 可折叠分析面板 |
| `{{STRATEGY_CARDS}}` | 操作建议 | 激进/稳健/防守策略卡片 |

## 🎨 常用 CSS 类

**颜色类：**
- `.text-rise` - 上涨红色
- `.text-fall` - 下跌绿色  
- `.text-highlight` - 强调橙色

**状态类：**
- `.rise` / `.fall` - 涨跌样式
- `.highlight` - 高亮数据
- `.limit-up` - 涨停闪烁效果

## 🚀 使用流程

1. 复制 `template.html`
2. 替换所有 `{{占位符}}`
3. 保存为新文件
4. 在浏览器中预览

**提示：** 查看 `template.html` 中的注释了解各组件的 HTML 结构。
